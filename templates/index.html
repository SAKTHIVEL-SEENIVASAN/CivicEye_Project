📌 What is CivicEye?

CivicEye is a simplified citizen–authority bridge. Citizens capture photos of civic issues, and the system automatically attaches GPS coordinates and timestamp, then emails the report to authorities.

⚙️ How It Works

1️⃣ Open website → Camera access.
2️⃣ Capture photo.
3️⃣ Auto geolocation + date/time.
4️⃣ Add description (optional).
5️⃣ Send → Email with image + location + details.

🛠️ Technologies Used

HTML5/CSS3 JavaScript MediaDevices API Geolocation API Flask (Python) SMTP Email

🎯 Key Features

  • ✅ One‑click camera capture
  • ✅ Auto geo‑tag & timestamp
  • ✅ Email with photo + details
  • ✅ Works on mobile/desktop
